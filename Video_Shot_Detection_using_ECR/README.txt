Video shot detection using Edge Change Ratio (ECR) values

Libraries used : NumPy, OpenCV for Python

Reference : "Automatic shot boundary detection combining color, edge, and motion features of adjacent frames" paper
            by A. Jacobs, A. Miene, G. T. Ioannidis, and O. Herzog
            
Instructions :

Before running the program read following points:
    1). Make a "Video" directory and put the videos in it for the input.
    2). Make the "Logs" directory to store logs for the analysis.
    3). Make the "Shot_Frames" directory to store the output shot frames.
